# Supported packages

For an always up-to-date package list please look [here](https://repos.managarm.org/repos/amd64/).

Managarm uses `xbps` for it's package management, users can use this to fetch packages from within managarm.

Some of the available packages include:

- Unix tools: `grep`, `sed`, `coreutils`, `diffutils`, `findutils`, `gawk`, `file`, `man-db`, `tar`, `which`, `groff`, `less`, `bzip2`, `gzip`, `xz-utils`, `lz4`, `zstd`, `bat`, `ripgrep`, `eza`
- Development: `yasm`, `nasm`, `patch`, `make`, `m4`, `gcc`, `cmake`, `binutils`, `bc`
- Scripting: `perl` (only basic functionality), `python`, `lua`
- Games: `ace`, `mednafen`, `tyr-quake`, `gtklife`, `nyancat`, `libremines`, `openttd`
- Networking: `wget`, `curl`, `socat`, `rsync`, `netcat`
- Communication: `hexchat`, `konversation`
- Editors: `nano`, `vim`
- Shells: `bash`, `dash`, `zsh`
- Graphical: `weston`, `mesa-demos`, `gtk+3`, `xclock`, `xkill`, `xwininfo`, `gtk4`
- Browsers: `minibrowser`, `links`
- Databases: `sqlite`, `gdbm`
- Bootloaders: `limine`
- Emulation: `qemu`
- Office tools: `kcalc`, `gnome-calculator`
- Init system: `systemd`
